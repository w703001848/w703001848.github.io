## 其他

1. 强调字体
   一个星号或者是一个下划线包起来，会转换为`<em>`倾斜，如果是2个，会转换为`<strong>`加粗

   ```markdown
   *md*    
   **md**
   _md_   
    __md__
   ```

    > 效果图：

    ![](img/13623636-c0fa033d2f13d33f.webp '强调字体')

   
   
2. 转义

   基本上和js转义一样,\加需要转义的字符

   ```markdown
   \\
   \*
   \+
   \-
   \`
   \_
   ```

3. 删除线

   用`~~`把需要显示删除线的字符包裹起来
   
   ```markdown
   ~~删除~~
   ```
