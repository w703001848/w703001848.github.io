## 细讲数据类型-列表

**定义：**[]内以逗号分隔，按照索引，存放各种数据类型，每个位置代表一个元素

#### 列表的增加操作

**追加**，数据会追加到尾部

```python
names.append("rain")
```

**插入**，可插入任何位置

```python
names.insert(2,"黑姑娘")
```

**合并**,可以把另一外列表的值合并进来

```python
names.extend(names2)
```

**列表嵌套**

```python
names[2] = [1,2,3]
names.insert(2,[1,2,3])
```
